# 🚀 DataForge Platform

> **Enterprise Data Engineering Platform built with Azure, Databricks, PySpark, SQL Server, Docker, Airflow and Power BI.**

---

## 📖 Overview

DataForge Platform is an enterprise-grade data engineering project designed to demonstrate how modern data platforms are built using cloud technologies and industry best practices.

The platform ingests data from multiple sources, stores it in a cloud-based Data Lake, processes it with Azure Databricks following the Medallion Architecture, serves curated data through SQL Server, and delivers business insights using Power BI dashboards.

Although the first implementation focuses on Sports Analytics, the architecture is designed to support multiple business domains such as Finance, Retail, Healthcare, Telecommunications, and Manufacturing.

## 🛠️ Technology Stack

- Python
- SQL Server
- Azure Data Lake Storage Gen2
- Azure Databricks
- PySpark
- Delta Lake
- Power BI
- Docker
- Apache Airflow
- Git & GitHub
